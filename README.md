# Project-
Создание парсера данных

1. Постановка задачи.
  Цель проекта - это создать парсер данных, принимающий уникальный идентификатор пользователя Russianfood, который извлекает различную необходимую информацию о находящихся там рецептах. Этот проект может быть полезен для сбора информации с различных сервисов, где нужно извлечь текстовые данные.
2. Инструкция по сборке и запуску кода.
   Для работы парсера вам потребуется установленный Python версии 3.6 или выше.
Установка:
- Скачать файлы  из репозитория на компьютер
- Установить библиотеки requests, beautifulsoup4 и pandas openpyxl
Запуск парсера:
Для запуска парсера нужно выполнить команду в терминале python: python ПРОЕКТ Парсинг.py
3. Пример результатов работы парсера.
  При успешном выполнении парсера вы получите таблицу. В комплекте есть пример (копия).
